Controls

Mac:

LEFT = CTRL + ALT + j
RIGHT = CTRL + ALT + l
JUMP LEFT = CTRL + ALT + i
JUMP RIGHT = CTRL + ALT + o

PC (chrome):

LEFT = ALT + j
RIGHT = ALT + l
JUMP LEFT = ALT + i
JUMP RIGHT = ALT + o

PC (firefox):

LEFT = SHIFT + ALT + j
RIGHT = SHIFT + ALT + l
JUMP LEFT = SHIFT + ALT + i
JUMP RIGHT = SHIFT + ALT + o
